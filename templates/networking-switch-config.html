Cisco Switch Configuration

Basic Switch Configuration using Cisco IOS CLI

CLI Configuration

enable
configure terminal
hostname SW1
enable secret admin123
service password-encryption
banner motd #
Unauthorized Access Prohibited
#
line console 0
password cisco
login
exit
line vty 0 4
password cisco
login
exit
interface vlan 1
ip address 192.168.1.2 255.255.255.0
no shutdown
exit
ip default-gateway 192.168.1.1
end
copy running-config startup-config

Command Explanation

Changes the hostname of the switch.

Sets the encrypted privileged EXEC password.

Assigns the management IP address to VLAN 1.

Verification Commands

Command Description
show running-config Displays the current running configuration.
show startup-config Displays the saved startup configuration.
show ip interface brief Displays interface IP addresses and status.
show vlan brief Displays VLAN information.
ping 192.168.1.10 Tests connectivity to PC1.

Security

Use strong passwords and encrypt them.

Save Config

Always save the running configuration.

Troubleshooting

Verify interfaces and IP settings before testing.
